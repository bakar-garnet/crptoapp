# crypto-oss-soup

A “busy” crypto demo repo that pulls in a lot of OSS deps across Node + Solidity + Python and runs a noisy workload in GitHub Actions.

## What it uses
- Node: ethers, viem, axios, zod, dotenv, chalk
- Solidity: hardhat, hardhat-toolbox, OpenZeppelin
- Python: web3.py, eth-account, requests, rich, python-dotenv
